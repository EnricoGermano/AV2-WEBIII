# AV2-WEBIII
Atividade de avaliação individual 2 da matéria de WEB III

## Manual de execução

### Pré-requisitos
Java 25 instalado


### 1 - Clonar o repositorio
```bash
git clone https://github.com/EnricoGermano/AV2-WEBIII.git
cd AV2-WEBIII
```
